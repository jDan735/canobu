# 🗿✂️📄 Canobu
Kanobu. Now rewritten in C++

# 🔨 Building
For building enter this lines:
```bash
cmake .
make
./canobu
```
## ⚠️ Recommendation
* If you use `MinGW` write:
```bash
cmake . -G "MinGW Makefiles" 
```
* If you use `Haiku x86` write:
```bash
cmake . -CMAKE_CXX_COMPILER=g++-x86
```
